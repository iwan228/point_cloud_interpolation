void IDWInterpolator::interpolate ( const vector< vector< Point > > & clouds,
vector< GridNode > & grid )

Performs IDW interpolation on the given grid using multiple point clouds.

This function interpolates weights for each node in a regular grid by considering the k nearest neighbors from each point cloud. It builds a KD-tree for each cloud and computes the weighted average based on the inverse of the squared distances.

Parameters
cloudsA vector of point clouds, where each point cloud is a vector of Point structures.
gridA reference to the vector of GridNode structures that will receive the interpolated values.
