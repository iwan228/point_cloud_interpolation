Generates a regular cubic 3D grid with a given step size and radius around the point (cx, cy, cz). 

Creates a three-\/dimensional grid within a cube of side length 2\texorpdfstring{$\ast$}{*}r and step size {\ttfamily step}. The center of the grid is specified by the coordinates (cx, cy, cz). This function is used to define a regular grid, which is then used for interpolation of values.
